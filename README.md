# RemoveVignette
Removes Vignette

### Installation
- Install [BepInEx](https://v-rising.thunderstore.io/package/BepInEx/BepInExPack_V_Rising/)
- Extract _RemoveVignette.dll_ into _(VRising folder)/BepInEx/plugins_

# RemoveVignette
Removes Vignette

### Installation
- Install [BepInEx](https://github.com/decaprime/VRising-Modding/releases/tag/1.690.2)
- Extract _RemoveVignette.dll_ into _(VRising folder)/BepInEx/plugins_
